Developer Workspace Kişiselleştirme

IDE ve Geliştirme Ortamı için 12 Farklı Modülü Aktifleştirin. (Veriler LocalStorage ile kaydedilir)

1. Git Auto-Commit

Değişiklikleri algılayıp otomatik olarak local branch'e commit atar.

2. Live Server Reload

DOM üzerinde bir değişiklik yapıldığında portu otomatik yeniler.

3. GraphQL Mocking

Backend API'leri hazır olmadan dummy verilerle GraphQL endpoint'i üretir.

4. JWT Auto-Refresh

Oturum tokenleri süresi dolmadan arkaplanda otomatik yenilenir.

5. Docker Auto-Build

Dockerfile güncellendiğinde container'i yeniden build eder.

6. Advanced Syntax Highlighting

Kod editörü bileşenlerinde derin syntax renklendirmesini açar.

7. Code Auto-Format (Prettier)

Dosya kaydedildiğinde standartlara göre kodu otomatik formatlar.

8. Webpack Bundle Analyzer

Derlenen Javascript dosyalarının boyut haritasını çıkarır.

9. Network Throttling (3G Modu)

Uygulamayı test etmek için bağlantı hızını yavaşlatır.

10. Service Worker Bypass

Geliştirme esnasında önbellekleme (Cache) sistemlerini devre dışı bırakır.

11. Headless Browser Mode

E2E testleri için arkaplanda görünmez bir Chromium sekmesi çalıştırır.

12. Strict Mode Linting
